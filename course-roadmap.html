Course Recommendations and Roadmap

Discover the best courses and a suggested path to excel in your Computer Science journey at Hunter College.

Recommended Courses

Here are some highly recommended courses that will build a strong foundation in Computer Science.

  • Intro to Computer Science: A beginner course that covers basic concepts and programming skills.
  • Data Structures: Learn the core structures and algorithms that are essential for efficient programming.
  • Computer Architecture: Understand how computers work at the hardware level.
  • Algorithms: Dive into complex problem-solving techniques and performance analysis.
  • Operating Systems: Gain insight into how operating systems function and manage resources.
  • Software Engineering: Learn best practices for designing, developing, and testing software applications.

Suggested Roadmap

This roadmap is designed to guide you through each year of your Computer Science degree:

First Year

  • Introduction to Computer Science
  • Basic Programming Concepts
  • Mathematics for Computer Science

Second Year

  • Data Structures
  • Discrete Mathematics
  • Computer Architecture

Third Year

  • Algorithms
  • Operating Systems
  • Database Management Systems

Fourth Year

  • Software Engineering
  • Electives in Advanced Topics
  • Capstone Project
